\doxysubsection{Opis szczegółowy}
Główna klasa okna aplikacji, dziedziczy po {\ttfamily wx\+Frame}. 

Klasa {\ttfamily \doxylink{class_main_frame}{Main\+Frame}} reprezentuje główne okno aplikacji, które umożliwia użytkownikowi interakcję z aplikacją do analizy danych o jakości powietrza. Zawiera wszystkie interfejsy użytkownika, takie jak pola tekstowe, listy, przyciski, oraz odpowiednie metody do obsługi zdarzeń.

Pobiera dane o jakości powietrza z serwera. 

Pobiera dane stacji pomiarowych z API GIOŚ.

Ta metoda wykonuje żądanie HTTP do serwera, aby pobrać dane o jakości powietrza. Po pobraniu danych uruchamiana jest analiza danych i ich wyświetlenie. W przypadku braku dostępu do internetu, próbuje załadować dane z pliku lokalnego.

Funkcja wykonuje zapytanie do serwisu GIOŚ w celu pobrania listy dostępnych stacji pomiarowych, które następnie są dodawane do rozwijanej listy stacji. W przypadku błędu wyświetlana jest odpowiednia informacja.

\begin{DoxyNote}{Nota}
W przypadku braku połączenia z serwerem GIOŚ lub problemu z przetwarzaniem danych wyświetlana jest wiadomość o błędzie. 
\end{DoxyNote}
