Inicjalizacja aplikacji. 

Funkcja inicjalizująca aplikację.

Funkcja {\ttfamily On\+Init} jest wywoływana przy starcie aplikacji. Tworzy główne okno aplikacji oraz ustawia jego właściwości (np. rozmiar, centrowanie).

\begin{DoxyReturn}{Zwraca}
Zwraca {\ttfamily true}, jeśli inicjalizacja aplikacji zakończyła się sukcesem, w przeciwnym razie zwraca {\ttfamily false}.
\end{DoxyReturn}
Funkcja {\ttfamily On\+Init} jest wywoływana przy uruchamianiu aplikacji. Tworzy główne okno aplikacji (obiekt klasy {\ttfamily \doxylink{class_main_frame}{Main\+Frame}}), ustawia jego rozmiar, centrowanie i wyświetlanie. Funkcja zwraca {\ttfamily true}, co oznacza, że aplikacja została poprawnie zainicjalizowana.

\begin{DoxyReturn}{Zwraca}
Zwraca {\ttfamily true}, jeśli inicjalizacja aplikacji powiodła się, w przeciwnym razie {\ttfamily false}. 
\end{DoxyReturn}
