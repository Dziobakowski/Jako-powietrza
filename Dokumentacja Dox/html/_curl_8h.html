Opis szczegółowy

Deklaracja funkcji do obsługi zapytań HTTP oraz parsowania odpowiedzi w formacie JSON.

Plik nagłówkowy zawiera deklaracje funkcji wykorzystywanych do wykonywania zapytań HTTP z wykorzystaniem biblioteki CURL oraz parsowania odpowiedzi w formacie JSON.

bool parseJsonResponse ( const std::string & response,
Json::Value & root )

Parsuje odpowiedź w formacie JSON.

Funkcja ta przyjmuje odpowiedź serwera w formie ciągu znaków i parsuje ją do obiektu typu Json::Value. Biblioteka JsonCpp jest wykorzystywana do konwersji odpowiedzi w format JSON.

Parametry
responseOdpowiedź serwera w formacie JSON w postaci ciągu znaków.
rootObiekt typu Json::Value, do którego zapisane zostaną dane z odpowiedzi.
Zwraca
Zwraca true, jeśli odpowiedź została poprawnie sparsowana, w przeciwnym razie false.

Parsuje odpowiedź w formacie JSON.

Funkcja ta konwertuje odpowiedź w postaci ciągu znaków (response) do obiektu typu Json::Value przy użyciu biblioteki JsonCpp. W przypadku błędu zwraca false.

Parametry
responseOdpowiedź w formie ciągu znaków, która ma być sparsowana.
rootObiekt typu Json::Value, do którego zapisane zostaną dane.
Zwraca
Zwraca true, jeśli odpowiedź JSON została poprawnie sparsowana, w przeciwnym razie zwraca false.

◆ performCurlRequest()

bool performCurlRequest ( const std::string & url,
std::string & response )

Wykonuje zapytanie HTTP do podanego URL.

Funkcja ta wykorzystuje bibliotekę CURL do wysyłania zapytań HTTP na określony adres URL. Odpowiedź serwera jest przechowywana w zmiennej przekazanej jako argument. Funkcja wyłącza weryfikację SSL.

Parametry
urlAdres URL, do którego wysyłane jest zapytanie HTTP.
responseZmienna, do której zostaną zapisane dane zwrócone przez serwer.
Zwraca
Zwraca true, jeśli zapytanie zakończyło się sukcesem, w przeciwnym razie false.

Wykonuje zapytanie HTTP do podanego URL.

Funkcja ta inicjuje sesję CURL, ustawia odpowiednie opcje (URL, funkcję callback do zapisu danych, wyłącza weryfikację SSL) i wykonuje zapytanie HTTP. Odpowiedź serwera jest zapisywana w zmiennej response.

Parametry
urlAdres URL, do którego zostanie wysłane zapytanie.
responseReferencja do zmiennej std::string, w której zostanie zapisany wynik zapytania.
Zwraca
Zwraca true, jeśli zapytanie zostało wykonane pomyślnie, w przeciwnym razie zwraca false.
